# Projet Finale Deramatica

This project is a deep learning platform for detecting dermatological diseases from skin images. It uses convolutional neural networks (CNN) to identify multiple skin conditions. Designed to support medical professionals in fast and accurate diagnosis.

---

## Features
- Upload and analyze skin images
- Detect and classify dermatological conditions
- Provides probability and confidence scores
- Intuitive web interface for users

---

## Requirements
- Python 3.9+
- TensorFlow / PyTorch
- OpenCV
- NumPy, Pandas, Matplotlib
- Flask or Django for web deployment
